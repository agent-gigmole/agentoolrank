## 2026-03-27 第一次讨论

### 起因

灵感线索 #10：做 AI 某个垂直方向的目录/导航网站，靠 SEO 吸引流量变现。

### 对标分析

研究了三家现有 AI 导航站：

| 网站 | 月访问量 | 估算月收入 | 特点 |
|------|---------|----------|------|
| theresanaiforthat.com | 780万 | $5-15万 | 按任务搜索，一个人做的，2023年1月上线第一个月15万访问 |
| toolify.ai | 510万 | $3-10万 | 中国团队，给工具标流量数据，56%流量来自Google SEO |
| futurepedia.io | 50-70万 | $0.5-2万 | 有付费课程$24.99/月，流量下降中 |

### 他们做对了什么

1. **theresanaiforthat** — 按任务搜索（"我要生成图片"而不是"图片工具列表"），用户视角
2. **toolify.ai** — 给每个工具标流量数据（月访问量），用户能看到哪个真的有人用
3. **futurepedia** — 做了付费课程（不只靠广告，多元变现）
4. **共同点** — SEO 是生命线（50-56%流量来自 Google 搜索）

### 他们的弱点（我们的切入点）

1. **工具太多太杂** — toolify 有 28000+ 个工具，用户根本选不出来
2. **没有深度评测** — 只有一句话简介，没有真正用过的对比
3. **没有工作流** — 只告诉你有什么工具，不告诉你怎么组合使用
4. **没有行业视角** — 按功能分类（图片/文字/视频），不按行业分（电商/金融/教育）
5. **数据更新靠 LLM 自动生成** — 质量参差不齐

### 讨论的四个切入方向

| 方向 | 优势 | 劣势 |
|------|------|------|
| A. AI Agent 工具导航 | 正在爆发期，大站覆盖不深，自己是重度用户 | 方向可能太宽 |
| B. AI 量化/金融工具 | 自己最懂 | 受众太窄 |
| C. AI + 电商工具 | 电商卖家付费意愿强 | 需要了解电商行业 |
| D. AI 开发者工具（MCP/Skills/Plugins） | 极度垂直，竞争最少，开发者付费意愿高 | 受众偏技术 |

### 关于"深度测评"的讨论

问题：不可能一个个亲自测评。

解决方案：不做"测评网站"，做"数据驱动的导航站"。内容生产自动化：

1. **爬取**（自动）：GitHub trending / Product Hunt / 各 marketplace → 工具信息
2. **LLM 生成**（自动）：喂 README + 文档 → Claude 自动写简介/优缺点/对比/上手步骤
3. **数据排序**（自动）：GitHub stars / npm 下载量 / 更新时间 / commit 活跃度
4. **人工**（20%）：定分类体系 + 偶尔审核质量

差异化不靠人工测评，靠：
- GitHub 活跃度排序（star 趋势 / 最近 commit）
- 社区热度（讨论量 / issue 活跃度）
- LLM 自动对比同类工具
- 更新频率（每天自动刷新，大站是每周）

核心洞察：**跟论文扫描一个套路 — 爬虫 + LLM + cron 自动化。**

### 技术方案（简单版）

```
Next.js + Tailwind → Vercel 免费部署
核心页面3个：首页（分类网格+搜索）/ 分类页 / 详情页（SEO长尾主力）
内容更新：cron + LLM 自动化
```

### 变现方式

- 付费置顶 / 推荐位（$50-200/月/工具）
- Affiliate 链接
- Google AdSense（流量起来后）
- 可以作为其他项目的获客入口

### 时间规划

- 第1周：搭基础版 + 初始内容填充
- 第2-3周：LLM 自动化内容管道 + SEO 优化
- 第4周+：等 SEO 发酵，持续自动更新
