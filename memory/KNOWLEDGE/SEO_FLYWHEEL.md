# SEO 飞轮建议

> 来源：2026-03-29 闲聊 session 讨论 + 竞品调研
> 核心问题：目前用户来了→看了→走了，没有增长闭环，纯靠 SEO 单腿走路

---

## 竞品飞轮分析

| 竞品 | 飞轮机制 | 效果 |
|------|---------|------|
| Futurepedia | ① 徽章换反链（工具方放「Featured by Futurepedia」徽章在官网）② 开发者自助提交工具 | 月活 100万+，15万注册用户 |
| There's An AI For That | 任务导向搜索（「我要做XX」→推荐→每个任务自动成为 SEO 页） | 每个用户搜索 = 一个长尾页面 |
| Toolify.ai | 自动爬取新工具 + 流量排名 + 工具间对比页 | 持续自动产生新内容 |

**共同点：都有「别人帮我生产内容」的机制，不是纯靠自己写。**

---

## 我们缺的 + 建议方案（按优先级）

### 1️⃣ Stack 分享页（最优先，飞轮效应最强）

**机制：**
- 用户用 AI Builder 生成 stack → 保存 → 生成唯一 URL（如 `/stack/abc123`）
- 用户分享到 Twitter/LinkedIn/团队 → 带流量回来
- 每个 stack 页面 = 一个可被 Google 索引的 SEO 页面
- stack 越多 → SEO 页面越多 → 搜索流量越大 → 新用户创造新 stack → 飞轮

**工程量：** 小（数据库加 stacks 表 + 分享页路由 + OG 图）

**飞轮公式：**
```
用户 → AI 生成 stack → 保存 → 分享 URL → 社交传播 → 新用户 → 新 stack → ...
                                                    ↓
                                              Google 索引 → SEO 流量
```

### 2️⃣ Tool A vs Tool B 对比页（Programmatic SEO）

**机制：**
- 自动生成「n8n vs Zapier」「LangChain vs LlamaIndex」等对比页
- 120 个工具 × 120 = 理论 14,400 个长尾页面
- 实际取同类别工具对比 ≈ 500-1000 个高质量对比页
- 用 LLM 批量生成对比内容（成本极低）

**工程量：** 中（批量生成脚本 + 对比页模板 + 路由）

**搜索量验证：** 「X vs Y」类搜索在 AI 工具领域搜索量很大（用户做采购决策时常搜）

### 3️⃣ 工具提交表单（让别人帮你生产内容）

**机制：**
- 开放 `/submit` 页面，开发者自己提交工具信息
- 审核后上线（或自动上线 + 人工后审）
- 开发者有动力提交：被收录 = 免费曝光

**工程量：** 小（表单 + 审核队列）

### 4️⃣ 徽章/嵌入组件（换反向链接）

**机制：**
- 给被收录的工具提供嵌入代码：`<a href="agentoolrank.com/tool/xxx">Featured on AgentoolRank</a>`
- 工具方放在自己官网 → 反向链接 → 提升 SEO 域名权重
- 类似 Product Hunt 的「Featured on Product Hunt」徽章

**工程量：** 极小（生成一段 HTML 代码即可）

### 5️⃣ 周报 Newsletter

**机制：**
- 每周自动生成「本周新增工具 + 热门 Stack + AI 行业趋势」
- 用 LLM 自动写，成本 ≈ $0.01/期
- 邮件订阅 → 用户定期回访

**工程量：** 中（邮件系统 + 自动生成 + 订阅管理）

## 建议执行顺序

| 顺序 | 项目 | 工程量 | 预期效果 | 依赖 |
|------|------|--------|---------|------|
| 1 | 徽章系统 | 极小（1天） | 中（换反链） | 无 |
| 2 | 工具提交表单 | 小（2天） | 高（免费内容） | 无 |
| 3 | Stack 分享页 | 小（3天） | 最高（飞轮核心） | AI Builder 已有 |
| 4 | 对比页 | 中（1周） | 高（长尾 SEO） | 工具数据已有 |
| 5 | Newsletter | 中（1周） | 中（留存） | 邮件系统 |

## AI Builder 输出升级（配合飞轮）

当前 AI 输出：工具列表 + JSON → 技术人看得懂，普通人不行

建议增加：**工作流故事**
- 每步标注用了哪个工具（【工具名】格式）
- 写成非技术人员能理解的用户场景描述
- 成本增加 ≈ $0.0003/次（可忽略）
- prompt 里加一段输出要求即可，DeepSeek 完全胜任

---

## 成本参考

- AI Builder 单次对话：$0.003（DeepSeek）
- 月 1万次对话：$30
- 对比页批量生成（1000页）：$5-10
- Newsletter 自动生成：$0.01/期
- **结论：AI 成本可以忽略，变现靠 affiliate 转化而非 AI 使用费**
